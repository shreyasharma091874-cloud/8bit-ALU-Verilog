# 8-bit Arithmetic Logic Unit (ALU)

This project implements an **8-bit Arithmetic Logic Unit in Verilog** capable of performing arithmetic and logical operations.

## Operations Supported

* Addition
* Subtraction
* AND
* OR
* XOR
* NOT
* Shift Left
* Shift Right

## Technologies

* Verilog HDL
* Digital Logic Design

## Files

alu.v → main ALU module
alu_tb.v → testbench for simulation
